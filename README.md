# Синхронизация: альтернатива тёмной материи

Анализ 85 галактик SPARC.  
**Формула:** `С = к · √(Сядра · П · Р)`

**П:** 1.2×10⁻¹⁰ м/с² — порог удержания (a₀ MOND - но другая интерпретация)

Небольшое уточнение. Уже после публикации препринта понял, что a₀ MOND не обязателен. Нужно было брать среднее значение от 85 галактив в анализе. Это 1.1×10⁻¹⁰ м/с². Но в целом на выводы это никак не влияет. Редактировать препринт не вижу смысла: наглядность совпадения a₀ MOND и среднего значения.

**Результат:** k падает с размером (r=-0.58, p=1.3×10⁻⁸)

**Запуск:**
1. `pip install astroquery pandas numpy`
2. `python analyze_sparc.py`
